
#newrelic-sysmond
================

This Ansible role installs
[NewRelic System Monitor](https://docs.newrelic.com/docs/server/).

Role Variables
--------------

    newrelic_sysmond:
      enabled: false    # change to true after you add the license key below
      license_key: ""   # NewRelic license key
